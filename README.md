# ATProto Export

Tool to export your ATProto account

## Usage

```
Usage: atproto-export [options] <handle-or-did>

Tool to export your ATProto account

Arguments:
  handle-or-did        Handle or DID to export

Options:
  -o, --out <out-dir>  Directory to export to (default: ".")
  -h, --help           display help for command
```
